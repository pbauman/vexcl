Constructor.

If host pointer is not NULL, it is copied to the underlying vector components of the multivector.

Parameters
commMPI communicator.
queuequeue list to be shared between all components.
nLocal size of each component.
hostHost vector that holds local data to be copied to the components. Size of host vector should be divisible by N. Components of the created multivector will have size equal to host.size() / N. The data will be partitioned equally between all components.
flagscl::Buffer creation flags.
