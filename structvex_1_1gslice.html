Detailed Description

template<size_t NDIM>
struct vex::gslice< NDIM >

Generalized slice selector.

This is very similar to std::gslice.

Index to base vector is obtained as start + sum(i_k * stride[k]), where i_k is coordinate along each dimension of gslice.
